# 🛕 DarshanEase

DarshanEase is a Temple Darshan Booking System developed using the MERN Stack.

## Features

- User Registration & Login
- JWT Authentication
- Temple Search
- Temple Details
- Online Darshan Slot Booking
- Booking Receipt
- My Bookings
- Cancel Booking
- Google Maps Integration
- Admin Dashboard
- Temple Management
- Slot Management
- Booking Management

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- CSS
- Axios

## Installation

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
