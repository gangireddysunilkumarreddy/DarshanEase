# 🛕 DarshanEase - Temple Darshan Booking System

## 📌 Project Description
DarshanEase is a MERN Stack based Temple Darshan Booking System that allows users to register, login, book temple darshan slots, and view booking history. The Admin can manage temples, slots, bookings, and users.

## 🚀 Features

### User
- User Registration
- User Login
- View Temples
- Book Darshan Slots
- Booking History
- Profile

### Admin
- Admin Login
- Add Temple
- Update Temple
- Delete Temple
- Manage Slots
- View Bookings
- Dashboard

## 🛠 Technologies Used

- MongoDB Atlas
- Express.js
- React.js
- Node.js
- JWT Authentication
- Axios
- CSS
