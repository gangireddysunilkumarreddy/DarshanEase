# 🛕 DarshanEase

## 📌 Project Overview

DarshanEase is a Temple Darshan Booking System developed using the MERN Stack. It allows users to browse temples, book darshan slots, manage bookings, and enables administrators to manage temples, slots, and bookings through a dedicated dashboard.

## ✨ Features

### 👤 User Features

* User Registration & Login
* Browse Temples
* Search Temples
* View Temple Details
* View Available Darshan Slots
* Book Darshan
* View My Bookings
* Cancel Booking
* Booking Receipt with Print Option
* Logout

### 👨‍💼 Admin Features

* Admin Login
* Dashboard with Statistics
* Add Temple
* Edit Temple
* Delete Temple
* View All Temples
* Add Darshan Slots
* View All Slots
* Delete Slots
* View All User Bookings

## 🚀 Installation

### Install Backend

cd server

npm install

npm run dev

### Install Frontend

cd client

npm install

npm run dev
